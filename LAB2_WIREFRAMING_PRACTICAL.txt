1. INTRODUCTION
================================================================================

This document presents the complete wireframing and UI flow planning for the
Smart Health Vault mobile application. The app provides secure health record
management for patients, doctors, and healthcare staff.

Wireframes represent the visual structure and layout of each screen without
final styling. The UI flow diagram shows navigation patterns and user journeys.

Tools Used:
- Figma (for wireframes)
- Draw.io (for UI flow diagram)

6. KEY DESIGN DECISIONS
================================================================================

1. ROLE-BASED DASHBOARDS
   - Different dashboards for Patient, Doctor, Staff
   - Customized content based on user role

2. CONSISTENT NAVIGATION
   - Bottom navigation for main sections
   - AppBar for screen-specific actions
   - Back button for hierarchical navigation

3. CARD-BASED LAYOUTS
   - Health records displayed as cards
   - Easy to scan and tap
   - Visual hierarchy with spacing

4. FORM VALIDATION
   - Inline error messages
   - Required field indicators
   - Clear success/error feedback

5. ACCESSIBILITY
   - Large touch targets (minimum 48x48dp)
   - Clear labels and icons
   - Readable font sizes

6. EMPTY STATES
   - Helpful messages when no data
   - Call-to-action buttons
   - Guidance for first-time users
